# MySQLToNeo4jMigrator
MySQLToNeo4jMigrator dient als Schnittstelle zwischen einer MySQL-Datenbank und einer Neo4j-Graphdatenbank. 

Im Kontext der Bachelorarbeit mit dem Titel "Graphenbasierte Kontextualisierung im Datenerfassungsprozess am Beispiel eines Materialflusses" dient das Programm MySQLToNeo4jMigrator dazu, Prozessdaten zur Visualisierung des Materialflusses einer Demonstrationsanlage zu kontextualisieren und eine solche Visualisierung in Form eines gericheten, azyklischen Graphen und eines Sankey-Diagramms zu erm√∂glichen.
